Getting Started For Building A Basic Web Application

Windows

  1. Install Git at http://git-scm.com/. Recommended installation options:
    • In the Selected Components section: leave as default: Windows Explorer integration, Git Bash Here, ...
    • In the Adjusting your PATH environment section: select Use Git from the Windows Command Prompt
    • In the Configure the line ending conversions section: select Checkout as-is, commit as-is
  2. Intall NodeJS at http://nodejs.org
    • Just click the install button on the site and run the downloaded .msi
    • Default install options should be enough
    • You must restart your computer once done installing

OS X

Git is installed by default on OS X. You will only need to install Node JS.

  1. Install NodeJS at http://notejs.org by clicking the green "Download" buton. Just execute the .pkg file and keep clicking "Next" until finished.
  2. Double-check that /usr/local/bin is in your $PATH variable. You can check this by opening a terminal window and typing in echo $PATH. If you need to add the path to your $PATH, you can follow this guide.
